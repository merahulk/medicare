# MedicareFrontEnd

A frontend web application built for managing Medicare-related operations with a clean and responsive UI.

## Tech Stack
- Angular 16
- TypeScript
- HTML5 & CSS3
- Maven

## Features
- Medicare data management interface
- Responsive and user-friendly design
- Component-based architecture

## Development Server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`

## Build
Run `ng build` to build the project.
